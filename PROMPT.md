이 프로젝트를 상세하게 분석해서 LLM 및 AI 어시스턴트들이 쉽게 해석할 수 있는 문서를 만드려고 해.

아래 규칙을 지켜서 작성해줘

- docs 폴더 생성
- docs 폴더 안에 각 항목별(인프라스트럭쳐, 디렉토리 구조, 파일 구조, 클래스 구조 등)로 파일을 분리해서 정리해줘
- PHP 패키지를 제외한 외부 라이브러리에 대한 설명이 필요해. jquery를 제외하고 ckeditor, jquery fileupload등은 이 프레임워크에 맞춰서 변경되어있어.
- 모듈, 애드온, 레이아웃, 모듈 스킨, 위젯, 위젯스타일, 에디터 컴포넌트와 같은 외부 서드파티 제작 가능 항목에 대해서도 정리가 필요해.

아래의 사항을 주의해줘

- 구조를 파악할땐 무조건 최우선적으로 파일을 읽고 추론을 진행해
- 20년 이상 넘은 오픈소스 프로젝트이며, 큰 구조가 바뀌지 않은 상태라 옛날의 사용방식과 현재 사용방식이 혼재되어있어.
- 다른 오픈소스 모델에서도 문제 없도록 문서를 꼼꼼하게 작성해줘
- 무조건 PHP 8.4를 기준으로 작성하고, 코드에서 deprecated로 mark된 부분들에 대한 설명은 필요없어
