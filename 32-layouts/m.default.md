# default (모바일 레이아웃)

## 개요

- **위치**: `m.layouts/default/`
- **목적**: Rhymix 기본 모바일 레이아웃. 햄버거 메뉴 + 콘텐츠 영역.

## info.xml 핵심

- `<menus>` 모바일용 GNB.
- `<extra_vars>`: LOGO_IMG, LOGO_TEXT 등.

## 구조

- 상단 고정 헤더 (로고 + 햄버거).
- 사이드 메뉴 (슬라이드 in).
- 콘텐츠 1단.
- 푸터.
