# user_layout (PC 레이아웃)

## 개요

- **위치**: `layouts/user_layout/`
- **목적**: 사용자 정의 레이아웃의 기반. 관리자가 시각 편집기로 자유 구성.

## 특징

- 코드 레벨 메뉴 슬롯이 거의 없음.
- 관리자가 위젯/HTML 블록을 드래그앤드롭으로 배치.
- 편집 결과는 `files/cache/layout/edited_<layout_srl>.html`로 저장.

## 관련

- layout 모듈: [../28-modules/layout.md](../28-modules/layout.md)
- 시각 편집기는 `modules/layout/`의 관리자 UI 참고.
